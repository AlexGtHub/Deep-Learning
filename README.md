# Deep Learning

This repository focuses on learning and building **Deep Learning** models using **PyTorch**, following a structured syllabus that progresses from basic neural networks to advanced topics and reinforcement learning.

---

##  Objective

To develop a solid understanding of deep learning concepts and implement them using PyTorch, from fundamental models to more advanced architectures.

---

## Syllabus

The content of this repository is based on the following topics:

1. **Deep Learning I: Introduction to Neural Networks**  
   - Basic concepts  
   - Perceptron  
   - Activation functions  

2. **Deep Learning II: Artificial Neural Networks (ANNs)**  
   - Fully connected networks  
   - Forward and backward propagation  
   - Loss functions and optimization  

3. **Deep Learning III: Convolutional Neural Networks (CNNs)**  
   - Convolutions and pooling  
   - Image classification  
   - Feature extraction  

4. **Advanced Neural Network Applications**  
   - Autoencoders  
   - Recurrent Neural Networks (RNNs)  

5. **Reinforcement Learning I: Multi-Armed Bandit Problems**  
   - Exploration vs exploitation  
   - Basic strategies  

6. **Reinforcement Learning II: Q-Learning**  
   - Value functions  
   - Policy learning  
   - Temporal difference learning  

---

##  Repository Structure (Branches)

The repository is organized using two main branches:

- **`main`** → Portfolio-ready projects  
  - Clean, well-structured implementations  
  - Documented code  
  - Reproducible results  
  - Suitable for showcasing  

- **`notebooks`** → Learning and experimentation  
  - Jupyter notebooks  
  - Concept explanations  
  - Prototyping and testing ideas  
  - Intermediate results and visualizations
